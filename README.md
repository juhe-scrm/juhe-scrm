## 安装

### 后端

1、cd backend-api

1、复制配置文件并修改 /conf/config.example.yaml -> /conf/config.yaml

2、go run .

### 前端

1、cd frontend-dashboard

2、npm -g install tyarn

3、tyarn install

4、tyarn start
